About

Medalist is an extension APP for the game Ingress Prime. By saving stats agents can win Medals and awards

Functions

The web APP Will need to have

-Use firebase as db

-Register and login user

-Register agent name

-Regist agent Stats (Multiple saves) - all Stats from the agent must be registered

-Awards system (badges like Ingress Prime)

-Profile page showing badges

-Events section

-Search section (players, Awards, events)
